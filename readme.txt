Como configurar os cookies do TikTok
===================================

Esta aplicação pode utilizar cookies do TikTok para realizar downloads ou uploads que exijam autenticação.

A) Via extensão de navegador (recomendado)
   1. Instale uma extensão como "cookies.txt" (Chrome / Firefox).
   2. Acesse tiktok.com e faça login normalmente.
   3. Clique no ícone da extensão → Export → salve o arquivo como cookies.json (ou cookies.txt, dependendo da extensão).

B) Manualmente pelo DevTools do Chrome
   1. Abra o Chrome em https://www.tiktok.com e faça login.
   2. Pressione F12 → aba Application → seção Cookies → selecione o domínio tiktok.com.
   3. Selecione todas as linhas de cookies, clique com o botão direito → Copy → Copy as JSON.
   4. Cole em um editor e salve como cookies.json.
